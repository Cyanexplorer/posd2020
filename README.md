# Homework Rule

1. 作業準時交，分數視通過測試

   (若無撰寫當次作業欲開發功能之單元測試，將酌量扣分)

2. Jenkins會於deadline準時關閉,作業於deadline後"兩天內"交出，以遲交計算，分數*70%

    (若有補交請寄信通知助教 "posd2020ta@gmail.com"，若無寄信將不予受理)

3. 作業於deadline後"兩天後"交出，將不予計分，此次作業0分計算
